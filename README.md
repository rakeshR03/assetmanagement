# assetmanagement

Created using java, spring boot, Rest API, Spring JPA, H2 database.

All the API links are available on "Asset Management.postman_collection.json" file.

Functionalities:
 1. Add Category
 2. Update Category
 3. List All Categories
 4. Add Asset
 5. List All Asset
 6. Get Asset by name
 7. Update Asset
 8. Assign asset to an employee
 9. Recover Asset
 10. Delete Asset
